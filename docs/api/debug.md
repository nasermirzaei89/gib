# debug API

## print(x, y, message)

```lua
debug.print(320, 240, "Hello")
```

Draws debug text in the render output at pixel coordinates.

Notes:

- Intended for quick diagnostics and overlays.
- For persistent game UI/text systems, build your own layer for now.
