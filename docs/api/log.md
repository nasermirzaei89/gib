# log API

Logging functions forward to Go slog output.

## debug(msg)

```lua
log.debug("debug message")
```

## info(msg)

```lua
log.info("hello")
```

## warn(msg)

```lua
log.warn("careful")
```

## error(msg)

```lua
log.error("failed")
```
